# Fiktivní Burza 📈📈📈

<!--Tento projekt vznikl 27. března 2026 v (zhruba) 9:00 ráno jako součást  transformace studenta SE na UTB.-->
Moderní finanční dashboard pro sledování akciového portfolia v reálném čase

## Funkce
- **Vejcoměr:** Sleduje počet zkonzumovaných bílkovin v reálném čase.
- **Live Data** Propojeno s API od Finnhub pro sledování cen akcií
- **Dynamické portfolio** Automatický výpočet majetku na základě pořízených akcií
- **Zachování dat** Ukládání do LocalStorage (zachování dat po zavření prohlížeče)
- **Design** Rezponzivní rozhraní s využitím Tailwind CSS.

## Tech Stack
- **Frontend** React.js (Hooks, Functional Components)
- **Design** Tailwind CSS (Grid, Flexbox, Animations...)
- **Icons** Lucide React
- **API** Finnhub API
- **Deployment** Vercel
